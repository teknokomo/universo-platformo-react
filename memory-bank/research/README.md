# Research Artifacts

This directory stores source-backed research documents created before or during PLAN mode.

Use this naming pattern:

```text
<topic-slug>-research-YYYY-MM-DD.md
<topic-slug>-research-YYYY-MM-DD-v2.md
```

Research artifacts must:

- Open and list every user-provided URL.
- Prefer primary sources over commentary.
- Record source freshness for unstable topics.
- Separate facts from inferences.
- Capture conflicts, uncertainty, project implications, and open questions.
- Link to the follow-up plan when planning begins.

The reusable template lives at:

```text
.agents/skills/research-before-plan/references/research-artifact-template.md
```
