# Flowise UI

🚨 **ПРЕДУПРЕЖДЕНИЕ О LEGACY КОДЕ** 🚨  
Этот пакет является частью устаревшей архитектуры Flowise и запланирован к удалению/рефакторингу после завершения миграции Universo Platformo (планируется на Q2 2026). Новые функции следует разрабатывать в современных пакетах `@universo/*`.

## Обзор

Основное React frontend приложение для интерфейса визуального программирования Universo Platformo, предоставляющее полный веб-интерфейс для создания, управления и развертывания AI рабочих потоков. Это основной пользовательский интерфейс, который интегрирует все функции Flowise с современными возможностями Universo.

## Информация о пакете

- **Пакет**: `flowise-ui`
- **Версия**: `2.2.8` (замороженная legacy версия)
- **Тип**: React Frontend Application (Legacy)
- **Фреймворк**: React 18 + Vite + Material-UI + React Flow
- **Паттерн**: Основное приложение, которое потребляет все остальные пакеты
- **Система сборки**: Vite с выделением 8GB памяти

## Ключевые особенности

### 🎨 Интерфейс визуального программирования
- **React Flow Canvas**: Визуальное программирование на основе узлов с drag-and-drop
- **200+ AI компонентов узлов**: Полная библиотека узлов LangChain и AI интеграций
- **Выполнение в реальном времени**: Живое выполнение рабочих потоков с потоковыми результатами
- **Система шаблонов**: Переиспользуемые шаблоны рабочих потоков и интеграция с marketplace

### 🏗️ Архитектура приложения
- **Мульти-представленческий интерфейс**: Редактор canvas, тестирование чатбота, управление учетными данными
- **Интеграция компонентов**: Использует `@universo/template-mui` для всех UI компонентов
- **Управление состоянием**: Интеграция Redux через `@universo/store`
- **Интеграция современных пакетов**: Включает все пакеты `@universo/*`

### 🌐 Функции платформы Universo
- **Аутентификация**: Современная система авторизации через `@universo/auth-frontend`
- **Пространства**: Управление рабочими областями через `@universo/spaces-frontend`
- **Публикация**: Публикация контента через `@universo/publish-frontend`
- **Метавселенные**: Интеграция виртуальных миров через `@universo/metaverses-frontend`
- **Конструктор пространств**: Создание 3D пространств через `@universo/space-builder-frontend`
- **Шаблоны**: Шаблоны викторин и интерактивных элементов

### 🔧 Функции разработки
- **Горячая перезагрузка**: Сервер разработки на базе Vite
- **Поддержка TypeScript**: Смешанный JS/TS с постепенной миграцией
- **Интернационализация**: Многоязычная поддержка через `@universo/i18n`
- **Интернационализированные метаданные**: HTML lang + мета-теги синхронизируются с i18n (runtime + build)
- **Интеграция API**: Централизованный API клиент через `@universo/api-client`

## Интернационализированные метаданные

Это приложение синхронизирует `<html lang>`, `<title>` и ключевые мета-теги с активным языком i18n:
- Runtime синхронизация происходит в `src/App.jsx`.
- Ранний `lang` задаётся inline-скриптом в `index.html` до загрузки React.

Список поддерживаемых языков подставляется при сборке:
- Исходный список: `@universo/i18n` `supported-languages.json`
- Vite HTML transform: `vite.config.js` заменяет `__SUPPORTED_LANGS__`
- `public/index.html` синхронизируется скриптом `scripts/sync-supported-langs.mjs` (prebuild)

Пример inline-фрагмента:
```html
const supportedLanguages = __SUPPORTED_LANGS__
```

## Интеграция API

### Паттерн useApi Hook
```javascript
import { useApi } from '@universo/template-mui/hooks'

function MyComponent() {
  const { request } = useApi(fetchList)
  
  useEffect(() => {
    request()
  }, [request])
}
```

### Интеграция API клиента
```javascript
import { useApiClient } from '@universo/api-client'

function DataComponent() {
  const api = useApiClient()
  
  const fetchData = async () => {
    const response = await api.get('/api/chatflows')
    return response.data
  }
}
```
