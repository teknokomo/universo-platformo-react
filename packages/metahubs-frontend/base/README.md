# MetaHubs Frontend

Frontend module for MetaHubs management in Universo Platformo.

## Overview

Provides UI components for managing metadata-driven entities (configurations):
- MetaHub list with card/table view
- MetaHub board with entity definitions
- Dynamic entity/field management
- User data records CRUD

## Features

- Responsive list/card views
- Real-time search and pagination
- Role-based access control
- Internationalization (en/ru)

## Usage

```tsx
import { MetahubList, MetahubBoard } from '@universo/metahubs-frontend'
```

## Development

```bash
# Build
pnpm build

# Watch mode
pnpm dev

# Lint
pnpm lint
```
