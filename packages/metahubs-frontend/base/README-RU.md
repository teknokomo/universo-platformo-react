# MetaHubs Frontend

Фронтенд-модуль для управления MetaHubs в Universo Platformo.

## Обзор

Предоставляет UI-компоненты для управления сущностями на основе метаданных:
- Список MetaHub с карточным/табличным представлением
- Доска MetaHub с определениями сущностей
- Динамическое управление сущностями/полями
- CRUD для пользовательских записей

## Разработка

```bash
# Сборка
pnpm build

# Режим наблюдения
pnpm dev

# Линтер
pnpm lint
```
