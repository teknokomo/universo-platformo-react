# MetaHubs Backend

Бэкенд-сервис для управления MetaHub в Universo Platformo.

## Обзор

MetaHubs предоставляет архитектуру, управляемую метаданными (MDA), для создания динамических сущностей и полей без DDL во время выполнения. Этот модуль позволяет пользователям определять собственные структуры данных через конфигурации, хранящиеся как метаданные.

## Архитектура

Модуль использует гибридный JSONB-подход:
- **sys_entity**: Определяет схемы сущностей (аналог таблиц БД)
- **sys_field**: Определяет спецификации полей для сущностей
- **user_data_store**: Хранит фактические записи пользователей в JSONB

## Ключевые возможности

- Динамические определения сущностей/полей через метаданные
- Хранение записей на основе JSONB для гибкости
- Поддержка RLS для мультитенантной безопасности
- Поддержка серверного UI через конфигурации представлений
