# Action Nodes

Action nodes represent operations, commands, or controlled state transitions.

![Metahub scripts as executable behavior surface](../../.gitbook/assets/quiz-tutorial/metahub-scripts.png)

## Typical Responsibilities

- Capture what the system or user can do.
- Connect business intent to execution logic.
- Describe explicit transitions instead of hidden side effects.

In Universo Platformo, action nodes are useful when structured models need explicit workflows, approvals, or automation.
