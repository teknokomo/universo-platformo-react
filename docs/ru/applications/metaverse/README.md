# Приложения «Метавселенные»

Метавселенная — это основной домен Universo Platformo. Он предоставляет способ организации проектов и миров в высокоуровневые "метавселенные" с контролем доступа, связями между метавселенными и пайплайнами публикации (например, шаблоны PlayCanvas MMOOMM).

## Статус: MVP

-   Реализован MVP бэкенда и фронтенда
-   Доступны операции листинга и создания со строгим RLS и аутентифицированным доступом

## Компоненты

-   Бэкенд: `@universo/metaverse-srv`
-   Фронтенд: `@universo/metaverse-frt`

## Возможности (MVP)

-   Список метавселенных, видимых пользователю (через членство или публичную видимость)
-   Создание метавселенной; создатель становится владельцем
-   Пункт меню слева → "Метавселенные" и маршрут `/metaverses`

## Безопасность

-   Per-request Supabase клиент с пользовательским JWT для RLS
-   PostgreSQL схема `metaverse` со строгими RLS политиками на всех таблицах

## Дорожная карта

-   Управление членством (роли), переключение метавселенной по умолчанию
-   Связи между метавселенными (дочерние/партнёрские/локация)
-   Интеграция импорта и публикации (MMOOMM PlayCanvas)
-   Обзорные страницы и представления ECS/UPDL
