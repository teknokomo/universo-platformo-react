# `packages/uniks-frt` — Uniks Frontend — [Статус: MVP]

Фронтенд приложение для управления рабочими пространствами (workspaces) в Universo Platformo.

## Назначение

Предоставляет пользовательский интерфейс для создания, организации и управления совместными рабочими пространствами.

## Ключевые возможности

- **Управление workspace CRUD**: Создание, чтение, обновление, удаление рабочих пространств
- **Управление участниками**: Добавление/удаление участников с управлением ролями
- **Многоролевая система**: Поддержка owner/admin/editor/member ролей
- **Responsive Material UI**: Адаптивный интерфейс
- **Интернационализация**: EN/RU локализация
- **Интегрированная навигация**: Встроенная в основную платформу

## Компоненты

### UnikList.jsx

Главная страница списка рабочих пространств с:
- Grid и list представлениями
- Поиском и фильтрацией
- Быстрым доступом к действиям
- Визуальными индикаторами статуса

### UnikDetail.jsx

Детальный вид рабочего пространства:
- Информация и настройки
- Список участников с управлением ролями
- Лента активности
- Быстрые действия

### UnikDialog.jsx

Модальное окно для операций:
- Создание нового workspace
- Редактирование свойств
- Конфигурация настроек
- Управление видимостью и правами

## Технологии

- **React**: Frontend framework
- **Material-UI**: Библиотека компонентов
- **i18next**: Интернационализация
- **TypeScript**: Типобезопасность

## Интеграция

### Навигация

Workspace routes монтируются под `/:unikId` префиксом с интеграцией в главное меню платформы.

### Аутентификация

Использует Supabase JWT токены для аутентификации пользователей.

## Разработка

```bash
# Установка зависимостей
pnpm install

# Сборка для продакшена
pnpm build --filter @universo/uniks-frt

# Режим разработки
pnpm --filter @universo/uniks-frt dev
```
