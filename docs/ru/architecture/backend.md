## Обзор

Бэкенд построен на Node.js с Express и использует TypeORM для доступа к данным. Каждая предметная область выделена в отдельный пакет бэкенда, зарегистрированный через общий ядро-бэкенд.

## Ключевые концепции

* Маршрутизация Express
* Паттерн репозитория TypeORM
* Мидлвары аутентификации
* Архитектура микросервисов
