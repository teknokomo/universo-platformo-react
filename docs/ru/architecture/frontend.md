## Обзор

Фронтенд построен на React с Material-UI (MUI) и использует Redux для управления состоянием. Каждая предметная область имеет свой фронтенд-пакет, который интегрируется через общее ядро фронтенда.

## Ключевые паттерны

* **Система шаблонов** — `universo-template-mui` (новый MUI-дашборд) и `flowise-template-mui` (легаси-сайдбар Flowise)
* **Загрузка данных** — `@tanstack/react-query` для управления серверным состоянием
* **Интернационализация** — `i18next` с локалями EN/RU
* **Маршрутизация** — React Router 6 с отложенной загрузкой маршрутов
