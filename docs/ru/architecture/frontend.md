# Архитектура фронтенда

Фронтенд строится вокруг `@universo/core-frontend`, который инициализирует
React-приложение и собирает функциональные пакеты в одну маршрутизируемую веб-оболочку.

## Паттерн рантайма

- React обеспечивает UI-рантайм.
- Vite используется для сборки фронтенда и dev tooling.
- Redux, React Query, auth helpers, routing и i18n компонуются на старте.
- Функциональные пакеты поставляют доменные страницы и UI-потоки.

## Текущая доменная поверхность

Активный публичный фронтенд включает authentication, onboarding, profile,
metahubs, applications, admin pages, shared templates и использование API client.

Поэтому фронтенд-оболочка — это модульная точка сборки, а не один жёстко
зашитый слой приложения.
