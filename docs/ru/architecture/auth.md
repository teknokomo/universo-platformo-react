## Обзор

Аутентификация реализована через Supabase Auth с поддержкой JWT-токенов. Система авторизации использует политики Row Level Security (RLS), позволяющие контролировать доступ к данным на уровне строк.

## Ключевые концепции

* Аутентификация через Supabase Auth
* JWT-токены и сессии
* Политики Row Level Security
* Управление ролями и разрешениями
