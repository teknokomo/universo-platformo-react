## Обзор

Universo Platformo использует PostgreSQL (через Supabase) с TypeORM для ORM. Все взаимодействие с базой данных осуществляется через паттерн репозитория; прямые вызовы к базе запрещены.

## Ключевые концепции

* **Единая база данных PostgreSQL** через Supabase
* **Сущности и миграции TypeORM** в каждом пакете
* **Центральный реестр миграций** в `flowise-core-backend`
* **UUID v7** для первичных ключей (сортируемые по времени)
* **Row Level Security (RLS)** для мультитенантной изоляции
