## Обзор

Universo Platformo — монорепозиторий, построенный на **pnpm workspaces** и **Turborepo**. Проект расширяет платформу Flowise AI многопользовательскими возможностями, фреймворком приложений и инструментами 3D/AR/VR-контента.

## Технологический стек

| Слой | Технология |
| --- | --- |
| **Среда выполнения** | Node.js 18+ |
| **Менеджер пакетов** | pnpm 9+ с workspaces |
| **Система сборки** | Turborepo, tsdown, tsc, Vite |
| **Бэкенд** | Express.js 4, TypeORM 0.3 |
| **Фронтенд** | React 18, TypeScript 5, Material UI 7 |
| **База данных** | PostgreSQL (Supabase) |
| **Аутентификация** | Passport.js + Supabase Auth |
| **Реальное время** | Colyseus WebSocket |

## Ключевые принципы

* Минимальные изменения в исходном коде Flowise
* Изоляция функциональности через независимые пакеты
* Паттерн TypeORM Repository для всего доступа к БД
* Общая DDL-логика времени выполнения в выделенных пакетах
