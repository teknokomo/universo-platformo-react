## Что такое Universo Platformo?

**Universo Platformo React** — это продвинутая платформа с открытым исходным кодом, построенная на основе [Flowise](https://flowiseai.com/), расширяющая его мощными возможностями для создания универсальных цифровых пространств, ИИ-потоков и иммерсивных 3D/AR/VR-приложений.

Платформа реализует **UPDL** (Universal Platform Description Language) — унифицированный узловой язык для описания 3D-сцен, игровой логики и интерактивного контента для различных движков и фреймворков.

## Ключевые возможности

* **Визуальный конструктор ИИ-агентов** — создание сложных ИИ-агентов через визуальный интерфейс
* **Система узлов UPDL** — универсальный язык описания для 3D/AR/VR-контента
* **Мультиплатформенный экспорт** — экспорт в AR.js, PlayCanvas, A-Frame и другие
* **Рабочие пространства (Юники)** — многопользовательское управление потоками
* **Приложения и метавселенные** — создание и управление интерактивными цифровыми мирами
* **Система публикаций** — публикация и распространение ваших творений
