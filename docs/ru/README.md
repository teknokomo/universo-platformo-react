# Добро пожаловать

Universo Platformo React — это текущая публичная эталонная реализация Universo
Platformo на React / Express / Supabase.

Эта документация построена как продуктовый GitBook: короткие посадочные
страницы, понятные разделы в сайдбаре и фактический контент, связанный с
текущим репозиторием.

## Что покрывает эта документация

| Раздел | Назначение |
| --- | --- |
| Начало работы | Безопасная установка, настройка, сборка и запуск репозитория. |
| Платформа | Описание текущего охвата платформы, доменных модулей и планов. |
| Архитектура | Описание монорепозитория, бэкенда, фронтенда, базы данных и аутентификации. |
| Справочник API | Описание публичной REST-поверхности и источника OpenAPI-документации. |
| Участие в проекте | Локальный процесс работы, правила кода и паттерны создания пакетов. |

## Текущее позиционирование продукта

Universo Platformo формируется как ERP-класс платформа с возможностями CMS для
организаций, цифровых сервисов, операционных процессов, пространств знаний и
будущих роботизированных производственных контуров.

Репозиторий в этом рабочем пространстве — самая полная публичная реализация
по запуску платформы, пакетным границам, миграциям, аутентификации, онбордингу,
профилям, metahubs, applications, администрированию и REST-документации.

## Как читать эту документацию

Начните с раздела «Начало работы», если хотите запустить репозиторий.

Переходите в «Платформу», если вам нужна продуктовая и доменная картина.

Используйте «Архитектуру» для технических фактов по репозиторию.

Используйте «Справочник API» для интеграции и «Участие в проекте» для правил.
