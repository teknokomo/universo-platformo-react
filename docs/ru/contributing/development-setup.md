# Настройка среды разработки

{% hint style="info" %}
**Страница в разработке.** Подробные инструкции по настройке будут добавлены в ближайшее время.
{% endhint %}

## Необходимые инструменты

* Node.js >= 18.15.0
* pnpm >= 9
* PostgreSQL (через Supabase или локальный)
* Git

## Шаги настройки

```bash
# Клонирование репозитория
git clone https://github.com/teknokomo/universo-platformo-react.git
cd universo-platformo-react

# Установка зависимостей
pnpm install

# Настройка окружения
cp .env.example .env
# Отредактируйте .env, указав параметры базы данных и сервисов

# Сборка всех пакетов
pnpm build

# Запуск серверов разработки
pnpm dev
```
