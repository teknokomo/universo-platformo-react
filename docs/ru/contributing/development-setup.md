# Настройка окружения

## Локальный процесс

1. Установите зависимости из корня репозитория через `pnpm install`.
2. Настройте локальные env-файлы бэкенда и при необходимости фронтенда.
3. Проверьте состояние через `pnpm build` из корня.
4. При необходимости запускайте точечные проверки пакетов и lint.

## Правила репозитория

- Предпочитайте root-level PNPM-команды.
- Используйте `pnpm --filter <package> lint` для проверки конкретного пакета.
- Избегайте `pnpm dev`, если вам не нужны live development servers.
- Держите English docs каноничными и зеркальте Russian docs по структуре.

Такой процесс сохраняет согласованность границ пакетов и generated artifacts.
