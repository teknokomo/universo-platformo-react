## Обзор

Мы приветствуем вклад в развитие Universo Platformo! Будь то исправление ошибок, добавление новых функций или улучшение документации — ваша помощь ценна.

## Начало работы

* [Настройка среды разработки](development-setup.md) — настройка локального окружения
* [Стандарты кодирования](coding-guidelines.md) — соглашения проекта
* [Создание пакетов](creating-packages.md) — добавление новых пакетов в монорепозиторий

## Быстрый старт для контрибьюторов

1. Сделайте форк репозитория
2. Создайте ветку для функции (`feature/your-feature`)
3. Внесите изменения, следуя стандартам кодирования
4. Отправьте Pull Request
