# Участие в проекте

Вклады приветствуются, но репозиторий всё ещё проходит через alpha-фазу, поэтому
полезная работа начинается с согласования с текущей архитектурой
и фактами репозитория.

## Лучшие направления для вклада

- Обновления документации, опирающиеся на текущие факты репозитория.
- Точечные исправления в существующих feature- или infrastructure-пакетах.
- Тесты, фиксирующие текущее поведение платформы.
- Архитектурная очистка, соответствующая SQL-first направлению рантайма.

Используйте страницы этого раздела до того, как откроете крупный набор изменений.
