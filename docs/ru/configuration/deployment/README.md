# Развёртывание

***

Universo Platformo спроектирована с платформенно-независимой архитектурой, обеспечивающей совместимость с широким спектром сред развёртывания в соответствии с потребностями вашей инфраструктуры.

## Локальная машина

Для локального развёртывания Universo Platformo следуйте нашему руководству [Начало работы](../../getting-started/).

## Современные облачные провайдеры

Современные облачные платформы отдают приоритет автоматизации и ориентированы на рабочие процессы разработчиков, упрощая управление облаком и текущее обслуживание.&#x20;

Это снижает требования к техническим знаниям, но может ограничить уровень настройки базовой инфраструктуры.

* [Elestio](https://elest.io/open-source/flowiseai)
* [Hugging Face](hugging-face.md)
* [Railway](railway.md)
* [Render](render.md)
* [Replit](replit.md)
* [RepoCloud](https://repocloud.io/details/?app\_id=29)
* [Sealos](sealos.md)
* [Zeabur](zeabur.md)

## Устоявшиеся облачные провайдеры

Устоявшиеся облачные провайдеры, с другой стороны, требуют более высокого уровня технических знаний для управления и оптимизации под ваши конкретные потребности.&#x20;

Однако эта сложность также предоставляет большую гибкость и контроль над вашей облачной средой.

* [AWS](aws.md)
* [Azure](azure.md)
* [DigitalOcean](digital-ocean.md)
* [GCP](gcp.md)
* [Kubernetes using Helm](https://artifacthub.io/packages/helm/cowboysysop/flowise)
