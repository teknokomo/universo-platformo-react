# Авторизация

***

Этот раздел познакомит вас с настройкой безопасности в Universo Platformo, с упором на механизмы аутентификации на уровне приложения и холста.

Внедряя надёжную аутентификацию, вы можете защитить свои экземпляры Universo Platformo и гарантировать, что только авторизованные пользователи смогут получить доступ и взаимодействовать с вашими холстами.
