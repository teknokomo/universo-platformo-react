## Обзор

Все запросы к API требуют аутентификации через Bearer-токены (JWT). Токены выпускаются Supabase Auth и должны передаваться в заголовке Authorization.

## Ключевые концепции

* Формат Bearer-токена
* Получение и обновление токена
* Области видимости и разрешения
* Ограничение частоты запросов (Rate limiting)
