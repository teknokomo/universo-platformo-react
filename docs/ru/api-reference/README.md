# Справочник API

Публичная API-поверхность в этом репозитории строится вокруг REST и
документируется живыми backend route packages вместе с отдельным пакетом `@universo/rest-docs`.

## Что покрывает этот раздел

- Базовый URL и форму смонтированной REST API-поверхности.
- То, как аутентификация меняет требования к запросам.
- Где искать и как запускать интерактивные OpenAPI- и Swagger-документы.

## Текущий охват

Текущий репозиторий раскрывает платформенные API для auth, locales, profile,
onboarding, applications, connectors, admin-потоков, public metahub routes и
design-time доменов metahub.

Используйте руководство по interactive docs, когда нужен сгенерированный current-state
inventory маршрутов вместо статического hand-maintained summary.
