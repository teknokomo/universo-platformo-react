# Справочник API

Публичная API-поверхность в этом репозитории строится вокруг REST и
документируется через модульные пакеты маршрутов вместе с пакетом `@universo/rest-docs`.

## Что покрывает этот раздел

- Базовый URL и форму публичной API-поверхности.
- Влияние аутентификации на API-запросы.
- Текущий статус webhook-подобных интеграций.

## Важная оговорка по охвату

Точный набор маршрутов зависит от того, какие функциональные пакеты смонтированы
в текущей сборке, но репозиторий уже раскрывает платформенные API для auth,
profiles, metahubs, publications, applications, admin-потоков и связанной конфигурации.
