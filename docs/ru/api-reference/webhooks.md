## Обзор

Вебхуки позволяют получать HTTP-уведомления в реальном времени при наступлении определённых событий в Universo Platformo. Вы можете подписаться на события для автоматизации рабочих процессов и интеграции с внешними системами.

## Ключевые концепции

* Регистрация вебхуков
* Формат полезной нагрузки (payload)
* Механизм повторных попыток
* Проверка подписи
