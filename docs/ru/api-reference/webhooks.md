# Вебхуки

Репозиторий пока не публикует стабильный публичный webhook-контракт.

## Текущий статус

Webhook-подобные точки интеграции со временем могут появляться в отдельных
модулях. В этой документации появится отдельный webhook-раздел, когда станет
доступен и поддерживаем версионируемый публичный контракт.

## Что использовать сегодня

Сейчас лучше опираться на документированную REST-поверхность, сгенерированную OpenAPI-поверхность и точки интеграции на уровне пакетов,
которые явно раскрыты в репозитории.
