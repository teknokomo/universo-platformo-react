# Начало работы

Этот раздел предназначен для разработчиков и технических ревьюеров, которым
нужно запустить репозиторий, изучить текущую реализацию или подготовить
локальное окружение для работы над изменениями.

## Кому стоит читать этот раздел

- Разработчикам, которые впервые поднимают монорепозиторий.
- Ревьюерам, проверяющим текущую реализацию платформы.
- Контрибьюторам, готовящим локальные изменения в пакетах или документации.

## Что важно до старта

Universo Platformo React всё ещё находится в альфе. Предпочтительна локальная
или тестовая установка, нужно ожидать архитектурные сдвиги и проверять
изменения через корневые команды сборки.

## Порядок чтения

1. Прочитайте страницу установки для зависимостей и начальной подготовки workspace.
2. Прочитайте страницу конфигурации для env-файлов и требований рантайма.
3. Прочитайте страницу быстрого старта для самого короткого безопасного пути к запуску.
