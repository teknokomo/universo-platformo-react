# Пространства

Пространства — это структурированные области платформы, которые группируют контент,
определения, шаблоны и операционный контекст вокруг одного домена.

![Рабочее пространство ресурсов для структурированного контента метахаба](../.gitbook/assets/entities/resources-workspace.png)

## Текущая роль

- Метахабы дают проектную структуру для типов сущностей, ресурсов и публикаций.
- Приложения дают рантайм-поверхности, синхронизированные из публикаций.
- Общие схемы и миграционные инструменты сохраняют воспроизводимость структурированного контента между окружениями.

## Рабочая граница

Используйте метахабы и ресурсы приложений как текущую границу проектирования и рантайма.
Не моделируйте пространства как отдельный публикуемый объект, если пакет явно не раскрывает такой контракт.
