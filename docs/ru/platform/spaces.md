## Обзор

Пространства (Spaces) — это контейнеры для визуальных холстов, где вы создаёте контент с помощью редактора на основе узлов. Каждый холст представляет собой рабочий поток или сцену, которую можно опубликовать и поделиться.

## Ключевые возможности

* Визуальный редактор с перетаскиванием
* Интеграция узлов UPDL для 3D/AR/VR-контента
* Версионность холстов и история изменений
* Совместная работа в реальном времени
