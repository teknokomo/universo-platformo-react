## Обзор

Space Builder — визуальный конструктор для создания и редактирования 3D/AR-пространств. Он предоставляет drag-and-drop интерфейс для работы с узлами UPDL, управления сценами и предварительного просмотра пространств в реальном времени.

## Ключевые возможности

* Визуальное редактирование сцены
* Drag-and-drop манипуляция сущностями
* Предварительный просмотр пространства в реальном времени
* Интеграция с граф-редактором UPDL
* Управление ассетами
