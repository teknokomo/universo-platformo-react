# Шаблон метахаба 1С-Совместимый

`1С-Совместимый` — опциональный шаблон метахаба для транзакционных учётных конфигураций.
Он предоставляет каталог пресетов метаданных для корпоративного учёта через общий Конструктор типов сущностей.

Этот шаблон не выбирается по умолчанию.
Новые метахабы по-прежнему стартуют с `Базового` шаблона, если пользователь явно не выбрал `1С-Совместимый`.

## Отсутствие аффилиации

Шаблон с использованием типов сущностей, совместимых с 1С:Предприятие 8.х.
Не является официальным продуктом 1С и не сертифицирован компанией 1С.
Реализация не импортирует и не копирует исходный код 1С, нативные файлы конфигураций, схемы баз данных, логотипы или UI-ресурсы.

## Каталог пресетов

Каталог зарегистрирован как метаданные шаблона и управляется через общий Конструктор типов сущностей:

| Пресет               | Роль для пользователя                                                                   |
| -------------------- | --------------------------------------------------------------------------------------- |
| Константы            | Значения верхнего уровня через переиспользуемый контракт поведения `singleValue`.       |
| Перечисления         | Существующий пресет Enumeration с представлением для шаблона.                           |
| Справочники          | Списки ссылочных данных с типизированным `catalogBehavior`.                             |
| Документы            | Транзакционные записи с типизированными `documentBehavior` и `documentPosting`.         |
| Журналы документов   | Метаданные журналов через `journalBehavior`, на первом этапе на основе `records.union`. |
| Регистры сведений    | Регистры фактов через `registerBehavior`.                                               |
| Регистры накопления  | Регистры остатков через `registerBehavior`.                                             |
| Планы счетов         | Иерархические планы счетов с флагами учета и связями субконто.                          |
| Планы характеристик  | Динамические характеристики для типизированной EAV-аналитики.                           |
| Регистры бухгалтерии | Метаданные дебет/кредит регистров с измерениями, ресурсами и проекциями.                |
| Планы видов расчета  | Метаданные зависимостей и вытеснения видов расчета.                                     |
| Регистры расчета     | Метаданные регистров расчета с периодами действия, базовыми периодами и перерасчетом.   |

Полный каталог включает Планы счетов, Планы видов характеристик, Регистры бухгалтерии, Планы видов расчета и Регистры расчета вместе с core runtime-пресетами. Их специализированные бухгалтерские и расчетные движки остаются preview-метаданными поведения, но сами типы сущностей и стартовые экземпляры доступны в новых 1С-совместимых метахабах.

## Контракт реализации

Шаблон должен оставаться metadata-driven:

-   специализированное поведение хранится в типизированных переиспользуемых секциях `catalogBehavior`, `documentBehavior`, `documentPosting` и `registerBehavior`;
-   сначала переиспользуются общие runtime widgets и примитивы DataGrid/FormDialog;
-   обычные пользовательские экраны не должны показывать raw UUID, raw JSON, `[object Object]`, внутренние имена таблиц или сырые сообщения валидации;
-   Константы в этом шаблоне являются сущностями верхнего уровня и не должны выглядеть для пользователя как дочерние элементы Наборов.

## Проверка

Для работ по шаблону ожидаются focused type/backend checks и Playwright coverage с локальной минимальной Supabase и тегом `@1c-compatible` перед расширением специализированного бухгалтерского или расчетного поведения за пределы preview-метаданных.
