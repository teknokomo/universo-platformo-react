# Навыки PlayCanvas Editor (Skills)

Навыки PlayCanvas Editor (PlayCanvas Editor Skills) — это специализированные руководства и промпт-системы, разработанные для ИИ-ассистентов (таких как Codex, Gemini, Claude и др.) для обеспечения безопасной, соответствующей требованиям и высокоструктурированной интеграции с PlayCanvas Editor в рамках платформы Universo.

Эти навыки представляют собой лучшие практики, собранные из официального руководства PlayCanvas и ограничений репозитория.

## 9 навыков PlayCanvas Editor

ИИ-ассистенты автоматически загружают или обращаются к этим навыкам во время разработки:

1. **`playcanvas-editor-interface`**
   - **Цель:** Руководство по навигации и манипулированию рабочей областью в интерфейсе PlayCanvas Editor, предотвращающее нарушение настроек макета или границ интерфейса редактора.

2. **`playcanvas-editor-api-realtime`**
   - **Цель:** Управление вызовами REST API и протоколом синхронизации в реальном времени ShareDB, обеспечение соответствия схемам данных и обработка авторизации.

3. **`playcanvas-editor-scenes`**
   - **Цель:** Управление переходами, загрузкой, выгрузкой и синтаксическим анализом иерархии сцен PlayCanvas без утечек памяти.

4. **`playcanvas-editor-assets`**
   - **Цель:** Создание реестра ассетов, асинхронная загрузка, сопоставление и очистка памяти GPU/RAM при уничтожении ассетов.

5. **`playcanvas-editor-scripting`**
   - **Цель:** Определение структуры скриптов сущностей (entities), корректное объявление хуков жизненного цикла (`initialize`, `update`, `postUpdate`, `destroy`) и безопасная горячая перезагрузка.

6. **`playcanvas-editor-version-control`**
   - **Цель:** Выполнение операций checkout, коммитов, ветвления и разрешения конфликтов в процессе работы с VCS PlayCanvas.

7. **`playcanvas-editor-settings`**
   - **Цель:** Валидация физики, слоев, параметров рендеринга вьюпорта и параметров запуска.

8. **`playcanvas-editor-authoring`**
   - **Цель:** Автоматизация сопоставления компонентов, построение иерархии сущностей и создание экземпляров шаблонов (templates).

9. **`playcanvas-editor-universo-compat`**
   - **Цель:** Ограничение агентов локальными портами и управлением токенами, предотвращение утечек безопасности при работе с PlayCanvas на платформе Universo.

## Руководство для разработчиков

При запросе к ИИ-ассистентам на изменение любого кода PlayCanvas Editor:
- Убедитесь, что агент прочитал соответствующий файл навыка в директории `.agents/skills/playcanvas-editor-*`.
- Проверяйте вывод кода на соответствие этим критериям перед коммитом изменений.
