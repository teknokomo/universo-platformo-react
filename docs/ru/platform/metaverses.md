# Контекст моделирования

Эта страница фиксирует только текущую границу репозитория для работы, связанной
с моделированием и виртуальными мирами.

## Текущая роль

![Рантайм-поверхность приложения](../.gitbook/assets/quiz-tutorial/runtime-quiz.png)

Работа, связанная с моделированием, сейчас опирается на те же контракты платформы, что и другие приложения:
идентичность, метахабы, публикации, синхронизацию приложений и рантайм-интерфейс.

## Что текущий репозиторий уже даёт

- Общая идентичность, членство и организационные границы.
- Проектные структуры через метахабы и публикации.
- Переносимые описания приложений и кросс-стековое мышление.
- Рантайм-поверхности приложений, которые потребляют опубликованные структуры.

Для проверки этого пути используйте текущую документацию по метахабам и приложениям.
