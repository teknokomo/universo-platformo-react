## Обзор

Метахабы служат хабами управления контентом, организующими публикации по веткам. Они обеспечивают структурированный способ управления версионным контентом для распространения.

## Ключевые возможности

* Организация контента по веткам
* Управление жизненным циклом публикаций
* Контроль версий опубликованного контента
* Интеграция с приложениями
