## Основные модули

Universo Platformo состоит из нескольких взаимосвязанных модулей:

| Модуль | Описание |
| --- | --- |
| **Рабочие пространства (Юники)** | Многопользовательское управление рабочими пространствами |
| **Пространства и холсты** | Визуальные редакторы на основе потоков |
| **Метавселенные** | Управление виртуальными мирами и их развёртывание |
| **Приложения** | Фреймворк бизнес-приложений с шаблонами |
| **Метахабы** | Управление контент-хабами с ветвлением и публикациями |
| **Система публикаций** | Экспорт и распространение контента |
| **Система узлов UPDL** | Универсальный язык описания для 3D/AR/VR |
| **Space Builder** | ИИ-генерация графов из текста |
| **Аналитика** | Аналитика использования контента |
