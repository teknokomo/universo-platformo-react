# Узлы действий

Action nodes представляют операции, команды или контролируемые переходы состояния.

## Типичные обязанности

- Фиксировать, что может сделать система или пользователь.
- Связывать бизнес-намерение с логикой исполнения.
- Описывать явные переходы вместо скрытых побочных эффектов.

В платформе вроде Universo Platformo action nodes особенно важны для сценариев,
потоков согласования, автоматизации и будущей координации агентов или роботов.
