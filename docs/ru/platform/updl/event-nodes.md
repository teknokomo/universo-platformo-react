# Узлы событий

Event nodes представляют триггеры, сигналы и наблюдаемые изменения, на которые
могут реагировать другие части системы.

## Типичные обязанности

- Отмечать, что некоторое событие произошло.
- Развязывать реакции с инициирующей операцией.
- Поддерживать мониторинг, автоматизацию и integration hooks.

Это делает event nodes релевантными для publications, applications, analytics,
уведомлений и будущих сценариев распределённой координации.
