## Обзор

Data-узлы управляют привязкой данных, внешними источниками и состоянием в виртуальных пространствах. Они позволяют подключать динамический контент и API к сущностям в пространстве.

## Типы данных

* Статические значения
* Привязки к API
* Реактивное состояние
* Внешние источники данных
