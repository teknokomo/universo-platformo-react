# Узлы данных

Data nodes несут значения, mappings, ссылки и типизированные payload,
которые затем потребляются другими nodes.

## Типичные обязанности

- Хранить структурированные значения или параметры.
- Связывать одну часть модели с другой.
- Делать data flow явным внутри более крупных графов.

В более широком контексте платформы data nodes помогают выражать конфигурацию,
промежуточные результаты, экспортируемые определения и payload для интеграций.
