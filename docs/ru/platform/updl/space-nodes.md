## Обзор

Space-узлы представляют корневые контейнеры для 3D/AR пространств в UPDL. Каждый Space-узел определяет окружение, параметры рендеринга и дочерние сущности внутри пространства.

## Свойства

* Параметры окружения
* Настройки рендеринга
* Дочерние узлы сущностей
* Параметры физики
