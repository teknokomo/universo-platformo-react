# UPDL

UPDL — это язык описания общих структур и переносимых определений.

В текущем направлении платформы полезнее рассматривать UPDL как более широкий
языковой слой для описания сущностей, связей, поведений и экспортируемых структур.

![Рабочее пространство сущностей как поверхность структурного моделирования](../../.gitbook/assets/entities/entities-workspace.png)

## Семейства узлов

- Узлы пространств задают ограниченные контексты и структурные контейнеры.
- Узлы сущностей описывают бизнес- или контентные объекты.
- Узлы компонентов добавляют переиспользуемые возможности или поля.
- Узлы действий выражают операции и переходы состояния.
- Узлы событий выражают триггеры и реакции.
- Узлы данных несут значения, связи и типизированные полезные нагрузки.

Нижележащие страницы по семействам узлов описывают роль каждой категории
в этом более широком направлении.
