## Обзор

UPDL (Universal Platformo Description Language) — декларативный язык на базе графа для описания виртуальных пространств и метавселенных. Он определяет структуру сцены, сущности, компоненты, действия, события и данные в виде типизированных узлов.

## Типы узлов

UPDL определяет семь базовых типов узлов:

| Тип узла   | Назначение                               |
| ---------- | ---------------------------------------- |
| Space      | Корневой контейнер 3D/AR пространства    |
| Entity     | Объект или набор объектов                |
| Component  | Визуальные / физические свойства         |
| Action     | Интерактивное поведение                  |
| Event      | Триггеры и условия                       |
| Data       | Привязка данных и внешние источники      |
