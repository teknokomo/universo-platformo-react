# Узлы сущностей

Узлы сущностей описывают главные объекты домена: людей, записи, активы,
документы, шаблоны, устройства и другие значимые единицы.

![Рабочее пространство сущностей для моделирования доменных объектов](../../.gitbook/assets/entities/entities-workspace.png)

## Типичные обязанности

- Определять идентичность доменного объекта.
- Раскрывать поля, связи или отношения владения.
- Служить опорой для действий, событий и компонентов.

Благодаря этому узлы сущностей полезны для бизнес-моделирования и
структурированного контента, а не только для графов интерактивных объектов.
