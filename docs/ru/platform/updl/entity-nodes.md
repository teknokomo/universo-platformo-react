# Узлы сущностей

Entity nodes описывают главные объекты домена: людей, записи, активы,
документы, шаблоны, устройства и другие значимые единицы.

## Типичные обязанности

- Определять identity доменного объекта.
- Раскрывать поля, связи или ownership-отношения.
- Служить опорой для actions, events и components.

Благодаря этому entity nodes полезны для бизнес-моделирования и
структурированного контента, а не только для графов интерактивных объектов.
