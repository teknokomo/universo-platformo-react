# Парсеры вывода

***

Узлы парсеров вывода отвечают за прием вывода модели и преобразование его в более подходящий формат для последующих задач. Полезно, когда вы используете LLM для генерации структурированных данных или для нормализации вывода моделей чата и LLM.
