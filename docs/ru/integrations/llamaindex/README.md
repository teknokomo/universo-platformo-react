# LlamaIndex

***

[LlamaIndex](https://www.llamaindex.ai/) — это фреймворк данных для приложений LLM, предназначенный для получения, структурирования и доступа к частным или специфичным для домена данным. Он обладает продвинутыми техниками извлечения информации для разработки RAG (Retrieval Augmented Generation) приложений.

Universo Platformo дополняет LlamaIndex, предоставляя визуальный интерфейс. Здесь узлы организованы в отдельные разделы, что упрощает создание рабочих процессов.

### Разделы LlamaIndex:

* [Агенты](agents/)
* [Модели чата](chat-models/)
* [Эмбеддинги](embeddings/)
* [Движок](engine/)
* [Синтезатор ответов](response-synthesizer/)
* [Инструменты](tools/)
* [Векторные хранилища](vector-stores/)
